# GiovanniClient 1.21.11


GiovanniClient is Closed source (no skidding) QoL mod for Hypixel Skyblock for modern minecraft versions.
It is the spiritual successor to [Polar Client](https://docs.polarclient.lol/polar/info/feature-list) , which featured a small list of "innocent cheats".

## Core Features

* **Auto Soulcry** (thanks  @HankSchneider
* **Auto Close hoppity eggs GUI** (thanks @HankSchneider)
* **Various rendering features** such as: NoFog, NoFireOverlay, NoBlindness, NoDarkness, NoHurtCam (...and more)
* **Disable Lightning Bolts** rendering
* **Show Invisible mobs** such as Shadow Assassins, Fels, etc. (includes an option to show EVERYTHING that's invisible)
* **Glowing** to starred dungeons mobs
* **Auto Agaricus Cap** for the Rift
* **Rift stuff**
* **(WIP) ESPs** for blocks, Corpses in the Glacite Tunnels and much more :)
